== Description ==

Some Feature

*Tracking the GoogleBot visited your site.

*Automatically send email once GoggleBot visited your site.
*Option to config or change email in setting menu.
*Simple config and usable.

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin
manually.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Setting, Chokelive Bottracker and config your E-Mail
4. Goto Appearance, Widgets and move 'Chokelive Bottracker' to sidebar or any where that you need.
5. Waiting GoogleBot visite your site and check trigger email, [also can see time update at widget as well]
